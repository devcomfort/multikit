## Environment Setup

Before running Ralph, ensure the environment is ready:

```bash
# Install tools (if not already)
.github/scripts/setup-copilot-ralph.sh --ralph

# Required environment variables
export COPILOT_GITHUB_TOKEN="${COPILOT_GITHUB_TOKEN}"
# Optional: export COPILOT_MODEL="gpt-4o"
# Optional: export RALPH_MAX_ITERATIONS=10
```

## Task Description

You are a CI governance reviewer operating in iterative mode.
Analyze this repository for compliance with the project's governance policies.
**Do NOT modify any files.** Only analyze and report.

Each iteration, pick the next unchecked category and verify it.

## Check Categories

### Iteration 1: README Compliance

- Verify README.md exists and contains required sections
- Check: title, description, installation, usage, license
- Verify badge URLs resolve (if present)

### Iteration 2: Project Structure

- Verify source and test directories exist
- Check configuration files are present and valid
- Look for orphaned or stale files

### Iteration 3: Version Consistency

- Find version declarations across all files
- Verify no conflicting versions
- Check CHANGELOG references latest version

### Iteration 4: CI Configuration

- Verify GitHub Actions workflows exist and have valid syntax
- Check secrets are referenced, not hardcoded
- Verify workflow triggers are appropriate

### Iteration 5: Final Report

- Compile results from all iterations
- Produce the final structured report (format below)
